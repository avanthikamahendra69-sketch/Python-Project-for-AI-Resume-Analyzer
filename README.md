## Project Overview
This project demonstrates an AI-based system that automatically screens resumes and ranks candidates based on how well their skills match a job description.

## Features
- Resume similarity scoring using NLP
- Skill extraction from resumes
- Skill match percentage calculation
- Candidate ranking
- Top 3 candidate recommendation
- Automated hiring recommendation
- Recruiter summary report

## Technologies Used
Python  
Scikit-learn  
TF-IDF Vectorization  
Cosine Similarity  

## Objective
Recruiters often receive hundreds of resumes.  
This AI system helps automate resume screening and quickly identify the best candidates using machine learning and NLP techniques.

## Future Enhancements
